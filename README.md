# README
 
项目内的几个翻译文件是来源于[yk47g](https://github.com/yk47g/gitkraken-chinese)的项目中。
 
但项目内使用的js比对是用了有道，我没有，也懒得申请。
 
所以使用现有技术做了一个小工具能直接比对替换指定的Json文件。
 
我使用的是9.11.0版本，替换的是9.5.1版本的，经过比对有128个字符是新增没替换的，重新手动把剩下的翻译了一下。如果有空再继续。
